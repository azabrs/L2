```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Программа выведет числа от 1 до 9, а после произойдёт deadlock. Причина в том, что (for n := range ch) конструкция будет бесконечно ожидать получение данных в канал, но так как горутина уже закончила выполнения данные никогда не будут отправлены.
```
