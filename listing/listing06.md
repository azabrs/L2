```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
3 2 3
Передавая слайс в функцию, мы можем изменять его значения. Но в тот момент, когда мы выполняем (i = append(i, "4")) cap превышает len и слайс перестраивается с новым базовым массивом, где cap в 2 раза больше предыдущего. После этого мы работаем с копией, и изменения в i больше не влияют на исходный срез.
```
